
> [!function]
> 显示文本的内容
> 






> [!grammer] 
```shell
less [参数] 文件

快捷操作：
enter     向下滚一行
空格键     向下翻一页
j         向下滚一行
k         向上滚一行
b         往回翻一页
f         往后翻一页
q         退出
/word     查找关键字
```



> [!example] 
```shell
less file.txt

# 查看多个文件,使用:n 浏览下一个，:p 浏览前一个
less file.txt file1.txt 
```
